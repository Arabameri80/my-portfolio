# My Portfolio

A personal portfolio website built with **Tailwind CSS**.  
This project is currently under development and will be completed step by step.

## Technologies Used

- HTML5
- CSS3 / Tailwind CSS
- JavaScript

## How to Run

1. Clone the repo
2. Install dependencies: `npm install`
3. Run the project: `npm run dev`
